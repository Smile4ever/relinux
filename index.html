Relinux

A free tool to create a professional-looking OS

About

Relinux is a free remastering tool for linux. It was originally written as a fork of the (now alive) remastering program named remastersys, but now it is completely rewritten in Python. The goal of this project is to let everyone from normal users to kernel hackers make their own OS easily.

Features

Fast

Relinux is completely multithreaded to let operations run at the same time, therefore decreasing the amount of time needed.

How to use

  1. Choose and install your base system
  2. Customize it
  3. Run relinux
